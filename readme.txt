Hi, it is instructions how to work with this project.
This project use nodejs, react, next, tailwind, jest, typescript, eslint, prettier

Run after you clone this project
1) Type to console "npm install"

For developmen
2) Type to console "npm run dev"

For Build
2) Type to console "npm run build"

For linting
2) Type to console "npm run lint"

For test
2) Type to console "npm run test"

This app work with Rest API and use Next SSR for reneder pages
